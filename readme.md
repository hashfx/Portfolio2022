## Implement Locally


### 1. Clone github repo

```bash
git clone https://github.com/hashfx/Portfolio2022.git
```

### 2. Install dependencies (just once)

```bash
npm install
```

### 3. Serve at localhost

```bash
npm run dev
```

### 4. Build for production in the dist/ directory

```bash
npm run build
```

### 5. Deploy to gh-pages

```bash
npm install gh-pages
npm run deploy
````
